# Chzzk VOD Downloader 

치지직(Chzzk)의 VOD 영상을 간편하게 다운로드할 수 있는 데스크톱 애플리케이션입니다.

## 주요 기능
- 치지직 VOD URL을 이용한 영상 다운로드
- 간편한 UI/UX
- FFmpeg를 이용한 고화질 영상 병합

## 사전 준비 사항
이 프로젝트를 로컬에서 실행하거나 빌드하려면 다음이 설치되어 있어야 합니다:
- [Node.js](https://nodejs.org/) (LTS 권장)
- [Rust](https://www.rust-lang.org/tools/install)
- Windows의 경우: [C++ Build Tools](https://visualstudio.microsoft.com/visual-cpp-build-tools/)

## 실행 및 빌드 방법

### 의존성 설치
```bash
npm install
```

### 개발 모드 실행
```bash
npm run tauri dev
```

### 배포용 빌드
```bash
npm run tauri build
```
빌드 결과물은 `src-tauri/target/release/bundle/` 경로에 생성됩니다.

## 사용 방법

1. **FFmpeg 설치**: VOD 다운로드를 위해 FFmpeg가 필요합니다. 앱 실행 시 상단에 나타나는 안내 배너의 **'설치'** 버튼을 클릭하여 자동으로 설치를 진행해 주세요.
2. **URL 입력**: 다운로드하고자 하는 치지직 VOD 또는 CLIP의 전체 URL을 입력창에 붙여넣습니다.
3. **저장 위치 선택**: '찾아보기' 버튼을 통해 영상이 저장될 폴더를 선택합니다.
4. **다운로드**: '다운로드 시작' 버튼을 누르면 다운로드가 진행됩니다. VOD의 경우 완료 후 자동으로 .mp4 파일로 변환 및 병합됩니다.

- **로그인 및 성인 인증**: 우측 상단의 톱니바퀴 버튼을 클릭하여 네이버 계정으로 로그인하거나 직접 쿠키 값을 입력할 수 있습니다. 성인 인증이 필요한 영상을 다운로드할 때 필요합니다.

> **참고**: 원활한 영상 변환을 위해 내부적으로 `ffmpeg`를 사용합니다. 앱 실행 시 안내 배너를 통해 설치하면 앱 전용 경로에 안전하게 구성되므로 별도로 파일을 찾아 설치할 필요가 없습니다.
